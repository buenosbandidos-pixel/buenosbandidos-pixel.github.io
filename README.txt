How to preview locally
- Open index.html in a browser, or run:
  python3 -m http.server 8000
  then visit http://localhost:8000

How to publish (fast)
- Netlify Drop:
  1) Zip this folder
  2) Drag the zip into Netlify “Deploy manually”
- Or any static hosting (GitHub Pages, Cloudflare Pages, etc.)
